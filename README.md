# Quote Collection

A community-focused quote collection to save and revisit quotes that inspire and resonate with everyone.

## About This Project

This is a simple web-based quote collection built as a learning project following the Vibe Coding Launchpad guide. The goal is to create a clean, accessible way to share and enjoy inspirational quotes.

## Current Status

✅ **MVP Complete** - Live on GitHub Pages!

### Milestones
- [x] Hello World - Repository setup and first commit ✅
- [x] MVP - Live site on GitHub Pages with quote display ✅
- [x] Interactive Features - Add Quote form with JavaScript ✅
- [ ] Additional features (localStorage, random quote button, delete, etc.)

## Features

- Display inspirational quotes with authors
- Interactive "Add Quote" form
- Clean, responsive design
- Mobile-friendly interface

## Tech Stack

- HTML/CSS/JavaScript (vanilla, no frameworks)
- Hosted on GitHub Pages
- Version control with Git/GitHub

## How to View

- **Local:** Open `index.html` in your browser
- **Live:** https://notaibot.github.io/launchpad/

## Project Goals

Keep it simple, ship often, and learn by building. This project follows the "ceramics rule" - make many small iterations rather than chasing perfection.
