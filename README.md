# Quote Collection

A community-focused quote collection to save and revisit quotes that inspire and resonate with everyone.

## About This Project

This is a simple web-based quote collection built as a learning project following the Vibe Coding Launchpad guide. The goal is to create a clean, accessible way to share and enjoy inspirational quotes.
